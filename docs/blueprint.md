# **App Name**: Diffraction Works

## Core Features:

- Project Showcase: Showcase open-source projects with descriptions, links to GitHub (https://github.com/Diffraction-Works), and example images.
- Contact with Links: Allow visitors to contact the photographer and developer via provided links (e.g., email, social media).
- Instagram Feed: Embed an Instagram feed from (https://www.instagram.com/diffraction_works/) to display recent photography.

## Style Guidelines:

- Primary color: Dark indigo (#3F51B5) for a professional, sophisticated look.
- Background color: Very dark gray (#121212), almost black, for a modern backdrop that emphasizes content.
- Accent color: Muted violet (#9575CD), desaturated to complement the primary color and add a subtle highlight to interactive elements.
- Body font: 'PT Sans' for a clean, readable design.
- Headline font: 'Space Grotesk' for a clean, readable design.
- Use clean, line-based icons to represent different sections and actions. Consider a photography-themed icon set.
- Employ a grid-based layout for image galleries and project showcases, ensuring responsiveness across devices.
- Implement subtle fade-in animations as images load or sections come into view, enhancing the user experience without being distracting.
